常见问题
需要什么编程基础？
需要 Python 基础（变量、函数、数据结构）和 RESTful API 开发经验。如果你写过 Flask/FastAPI/Express 做过 CRUD，基础就够了。不需要 ML/DL 经验，课程从 Transformer 原理讲起。
和其他 AI 课程有什么区别？
市面上的 AI 课要么只教理论（Andrew Ng 风格），要么只教 API 调用。我们从 Prompt Engineering 开始，一路做到 Fine-Tuning Llama 3.1 + Multi-Agent + MCP 协议，7 个实战项目全部可部署到生产环境。重点是工程化落地，不是 Notebook 跑 Demo。
需要 GPU 吗？
Phase 1-3 不需要 GPU，用 API 调用就行。Phase 4 Fine-Tuning 部分用 Google Colab (免费 T4 GPU) 或 AWS EC2 GPU 实例。课程会教你如何用 Quantization + QLoRA 在消费级硬件上微调模型。
ISA Capstone Project 是什么？
Intelligent Study Assistant — 一个贯穿全课程的实战项目。Week 1 用 Prompt 搭基础版，Week 4-6 加 RAG 支持文档检索，Week 7-9 加 Agent 能力（工具调用、多轮推理），Week 10-12 用 Fine-Tuning 优化。每个 Phase 结束时你的 ISA 都会升级一次。
MCP 是什么？为什么要学？
Model Context Protocol — Anthropic 提出的标准协议，用来连接 AI 模型和外部工具/数据源。你构建 MCP Server 后，Claude Code、Cursor 等 AI 工具都能直接调用。2025 年 AI 工程师 JD 里出现频率很高。
课程结束后能做什么？
你能独立构建 RAG 系统、Agent 应用、MCP Server，并部署到生产环境。简历上有 7 个可 Demo 的项目。目标岗位：AI Engineer、LLM Engineer、ML Platform Engineer、AI SaaS Developer。
课程是中文还是英文？
直播用中文授课，代码和文档用英文（和真实工作一致）。课程标题和技术术语保留英文原名，降低你看英文文档的门槛。
