Project: Production-grade Agentic RAG

直播
📖 教学内容

Agentic RAG vs Naive RAG：
- Naive RAG: Query → Retrieve → Generate（固定管线）
- Agentic RAG: Agent 自主决策何时检索、用什么工具、是否需要追问

路由决策：Agent 根据 query 类型选择不同策略
错误恢复：检索失败时的 fallback 机制
生产化：Rate limiting、Caching、Error handling、Logging

Multi-Agent Architectures

直播
📖 教学内容

三种多 Agent 架构：
1. Hierarchical — Boss Agent 分配任务给 Worker Agents
2. Collaborative — Agent 之间平等协作，共享上下文
3. Supervised — Supervisor 监控 Agent 行为，可以打断和修正

何时用 Multi-Agent：任务复杂度决策矩阵
状态管理：Agent 之间如何传递上下文和中间结果

9.1

Model Context Protocol (MCP)

直播
📖 教学内容

MCP 是什么 — Anthropic 提出的标准化协议，让 AI 模型可以安全地连接外部数据源和工具

MCP 架构：Client (Claude/Cursor) → Server (你的工具) → Resources/Tools/Prompts

为什么 MCP 重要：每个 AI Agent 都需要连接外部世界，MCP 是 2025 年的标准方式

源码解读：MCP TypeScript SDK 核心源码分析

🔬 Lab 实操

3. 在 Claude Code / Cursor 中连接并测试
