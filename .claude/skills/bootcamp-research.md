# /bootcamp-research — Bootcamp 市场调研与竞品分析

对一个 bootcamp 方向做完整的市场调研，包括岗位需求、薪资数据、竞品课程、目标用户画像。

## 使用方法
```
/bootcamp-research [方向名称]
```
例如：`/bootcamp-research "AI Adoption Specialist"` 或 `/bootcamp-research "AI Engineer"`

## 执行步骤

1. **岗位市场调研**：
   - 搜索 LinkedIn / Seek / Indeed 上的相关 JD
   - 提取高频技能要求（出现在 X/10 份 JD 中）
   - 整理薪资范围（Entry / Mid / Senior）
   - 识别常见 Job Title 变体（一个技能可能藏在不同 title 下）
   - 分析地域差异（澳洲 vs 全球）

2. **竞品课程分析**：
   - 搜索同类课程（Coursera / Udemy / General Assembly / 大学项目）
   - 对比维度：价格、时长、是否有实操、是否有场景教学、是否有职业支持
   - 找出竞品的**缺口**（他们没教但 JD 要求的）
   - 这些缺口就是我们课程的差异化卖点

3. **目标用户画像**：
   - 谁会来学这个课？（背景、痛点、目标）
   - 至少定义 3-5 个 persona
   - 每个 persona 的学习路径可能不同

4. **技能 → 课程映射**：
   - JD 要求的每个技能，我们课程哪节课覆盖了？
   - 有没有 JD 要求但我们没教的？
   - 输出 JD Gap Analysis 表

5. **输出格式**：
   - 市场数据摘要（不要 AI 味的空话，要具体数字和来源）
   - 竞品对比表
   - 用户 Persona 卡片
   - JD 技能覆盖度表
   - 课程差异化定位建议
