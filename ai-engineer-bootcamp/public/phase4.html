Week 11 · LLM Fine-Tuning: PEFT + QLoRA 直播 + 项目
11.1

Fine-Tuning 基础

直播
📖 教学内容

Fine-Tuning 的三大角色：
1. Style/Format — 控制输出风格和格式
2. Knowledge — 注入领域知识
3. Capability — 增强特定能力（代码、推理）

何时 Fine-Tuning vs RAG vs Prompt：
- RAG: 知识更新频繁、需要引用来源
- Fine-Tuning: 固定模式、低延迟、特定风格
- 两者结合: Fine-Tuned Model + RAG

🔬 Lab 实操

Downloading Open-Source Model Weights：
- HuggingFace Hub 模型下载和管理
- Model Card 怎么看、License 怎么选

Loading LMs on GPU：
- GPU 显存计算：参数量 × 精度 = 显存需求
- FP32/FP16/BF16/INT8/INT4 的区别
- Google Colab T4 能跑多大的模型

11.2

PEFT + Quantization + QLoRA

直播
📖 教学内容

PEFT (Parameter Efficient Fine-Tuning)：
- Full Fine-Tuning vs LoRA vs QLoRA
- LoRA 原理：低秩分解，只训练 A×B 矩阵
- rank、alpha、target_modules 参数选择

Quantization 量化：
- GPTQ、AWQ、bitsandbytes
- 4-bit NormalFloat (NF4) — QLoRA 核心

QLoRA = Quantization + LoRA：
在 4-bit 量化模型上做 LoRA 微调 — 消费级 GPU 可用

PEFTbitsandbytesHuggingFace Transformers
🔬 Lab 实操

Project: Fine-Tuning Llama 3.1 with PEFT-QLoRA

在 Google Colab 或 AWS EC2 上：
1. 加载 Llama 3.1 8B (4-bit quantized)
2. 准备训练数据（Week 10 生成的 SDG 数据）
3. 配置 LoRA (rank=16, alpha=32)
4. 训练 3 epochs
5. 评估：perplexity、task-specific metrics
6. 合并权重并导出

👨‍🏫 Fine-Tuning 结果 Review
